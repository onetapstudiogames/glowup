# Glow Up

GitHub Pages files for the Glow Up guides.

## Files
- `index.html` — landing page
- `eats.html` — Glow Up Eats
- `workout.html` — Glow Up Workout

## Publish on GitHub Pages
1. Upload these files to the root of the `main` branch.
2. In GitHub: **Settings -> Pages**.
3. Set **Source** to **Deploy from a branch**.
4. Choose **main** and **/(root)**.
5. Save.
